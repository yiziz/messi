Poisson Distribution Interactive Calculator

Distribution graph

Description

For the Poisson distribution, this calculator gives the value of the cumulative distribution function \(p = F(x)\) for a given value of \(x\), or the value of the quantile function \(x = F^{-1}(p)\) for a given value of \(p\). The Poisson parameter \(\lambda\) can be varied with the input controls, as can the variables \(x\) and \(p\)
